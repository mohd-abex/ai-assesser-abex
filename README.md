# InterviewAI Monorepo

This monorepo contains the InterviewAI application consisting of:

- `frontend/` – Next.js 14 App Router app (TypeScript, Tailwind, shadcn/ui)
- `ai-service/` – FastAPI service for AI (STT/LLM/TTS) orchestration
- `supabase/` – Database migrations, seed, and SQL utilities
- `docs/` – Product and development documentation

Follow `../docs/development_topology.md` to build in the recommended order.
